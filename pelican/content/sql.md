Select Clauses
==============
LIMIT count [OFFSET skip]  
count: how many rows to return.  
skip: how far into the result to start.  

ORDER BY columns [DESC]  
columns: which columns to sort by. Seperated by comas.  
DESC: sort in reverse order (descending).
