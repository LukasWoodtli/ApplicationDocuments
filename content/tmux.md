# Prefix Key

The prefix key combination needs to be pressed before any command for tmux (when tmux is running).

By default the prefix is:

`CTRL + b`


# Sessions

| Commands             | Function               |
|----------------------|------------------------|
| `tmux new -s <name>` | Create a named session |
| `tmux ls`            | List current sessions  |
